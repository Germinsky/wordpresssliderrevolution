## Image Creation Guidelines

### For Overlay PNGs:
- **Size**: 1920x800px (full HD width, slider height)
- **Format**: PNG-24 with transparency
- **File Size**: Target <200KB (use compression)
- **Elements**: Stage lights, curtains, equipment
- **Transparency**: 50-70% opacity for blend-ability
- **Colors**: Match theme aesthetic
- **Save**: "Save for Web" in Photoshop

### For Preview JPGs:
- **Size**: 300x200px (thumbnail)
- **Format**: JPG (quality 80)
- **File Size**: Target <50KB
- **Content**: Show the theme in action
- **Brightness**: Ensure readability in UI

## Image Processing Steps

1. **Find/Create Base Image**
   - Download from stock sites
   - Take photos at venues
   - Create in 3D software

2. **Edit in Photoshop/GIMP**
   - Resize to 1920x800px
   - Adjust brightness/contrast
   - Add lighting effects
   - Apply blur where needed
   - Set opacity to 60-70%

3. **Save as PNG**
   - File → Export → Save for Web
   - PNG-24 format
   - Enable transparency
   - Optimize file size

4. **Create Thumbnail**
   - Resize to 300x200px
   - Save as JPG
   - Quality: 80%

5. **Upload to Plugin**
   - Place in /assets/images/
   - Ensure filenames match exactly
